## Usage

1. Clone this repository

```sh
git clone https://github.com/vague-theme/vague.vim.git ~/.vim/pack/vendor/start/vague.vim
```

2. Modify your `vimrc` config file:

```vim
set termguicolors " Ensure this option is set
colorscheme vague
```
